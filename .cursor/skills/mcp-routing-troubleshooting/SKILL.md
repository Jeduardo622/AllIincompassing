# MCP Routing Troubleshooting

## Quick Start

1. Identify which MCP servers are colliding.
2. Apply the routing fix for the target server.
3. Re-run the failing MCP action.

## Steps

- Follow `docs/MCP_ROUTING_TROUBLESHOOTING.md`.
- Use repo script:
  - `scripts/mcp-routing-fix.js`
- Confirm the intended MCP server is active.

## Output

- Short summary of the conflict and fix applied.
- Any required follow-up steps.
